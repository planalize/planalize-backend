# Planalize Backend

Rest API for planalize

## Requirements

- [Node.js 18](https://nodejs.org/en/download/)
- [Docker](https://www.docker.com/)
- [pnpm](https://pnpm.io/)

## Installation

```bash
$ pnpm install
```

## Building the app

```bash
$ pnpm build
```

## Running the app

```bash
# development
$ pnpm dev

# production
$ pnpm start
```
